# Engineering Calculator

Инженерный калькулятор с поддержкой тригонометрии, логарифмов, факториала и памяти.

**[Открыть калькулятор →](https://subrosa5.github.io/calculator-web/)**

---

## Возможности

- **Темы оформления** — иконка ⚙ в углу дисплея открывает панель настроек: переключение светлой/тёмной темы и 5 цветовых схем кнопок (Space, Forest, Sunset, Ocean, Violet), выбор сохраняется в браузере
- **График функций** — визуализация f(x) на холсте: zoom колёсиком мыши, перемещение перетаскиванием, координаты точки при наведении
- **Тригонометрия** — sin, cos, tan, asin, acos, atan (режимы DEG / RAD)
- **Логарифмы** — log₁₀, ln
- **Степени и корни** — xʸ, x², √x, ∛x
- **Специальные функции** — n!, 1/x, |x|
- **Константы** — π, e
- **Память** — MC, MR, M+, M−
- **История вычислений** — клик по записи возвращает выражение
- **Live preview** — результат обновляется во время набора
- **Клавиатура** — полная поддержка ввода с клавиатуры (`p` → π)

## Клавиатуры

| Клавиша | Действие |
|---|---|
| `0–9`, `.` | Цифры |
| `+` `-` `*` `/` | Операторы |
| `^` | Степень |
| `%` | Остаток |
| `(` `)` | Скобки |
| `!` | Факториал |
| `p` | π |
| `Enter` или `=` | Вычислить |
| `Backspace` | Удалить символ |
| `Escape` | Очистить всё |
| `Delete` | Очистить выражение |
